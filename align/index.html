Display y Alineación de HTML

Introducción a la alineación básica de elementos HTML

Display de elementos

Antes de hablar de alineación tenemos que tener en cuenta los 3 tipos de elementos de html mas importantes en cuanto a como se muestran: Los elementos en bloque, los elementos en línea y su fusión inline-block

Elementos de tipo "block"

  • Ocupan todo su contenedor y se apilan uno sobre el otro
  • La altura varía dependiendo de su contenido
  • Pueden contener dentro otros elementos "block" o "inline"
  • Se puede modificar el ancho y el alto por CSS
  • Algunos de los elementos en bloque mas utilizados son los div, p h1, h2, h3, h4, h5, ul, ol y li

Elementos de tipo "inline"

  • Se posicionan horizontalmente uno al lado del otro.
  • No ocupan todo el ancho de su contenedor padre
  • Solo pueden contener elementos de tipo inline dentro
  • No se puede modificar el ancho y el alto por css
  • Algunos de los elementos en bloque mas utilizados son los a, span, input, button, textarea

Elementos de tipo "inline-block"

Tenemos un tercer tipo de elemento que es llamado 'inline-block', el cual es una combinacion de las propiedades de los anteriores

  • Sigue el flujo del texto como los elementos inline
  • Se puede modificar el tamaño con CSS
  • Las imagenes se comportan de esta manera

Alineación de elementos

Como ya mencionamos hay diferentes "tipos" de elementos HTML, y por cada uno de los ellos vamos a tener que tener en cuenta que por las propiedades de cada uno, no responden a los estilos de la misma forma. Por este motivo, para centrarlos recurriremos a distintos metodos para distintos tipos

Centrado de elementos en bloque

Uso de la propiedad align:center

Para el centrado de un elemento en bloque (y los elementos dentro de él), podemos utilizar la propiedad "text-align:center". En este caso se la asignamos a la clase "align-center". Esta propiedad sirve para centrar texto o elementos dentro del tag.

Uno de los elementos inline-block por excelencia son las imagenes. Los tags img se comporta como un elemento inline, pero ademas se puede modificar su ancho y su alto con CSS. Por comportarse como un estilo inline, no podremos aplicar el centrado con margin, pero se centrará si tenemos un padre con text-align:center

La propiedad "display"

Podemos modificar el comportamiento que va a tener un elemento dentro de nuestro HTML. Para esto podemos utilizar la propiedad "display". Como modo de ejemplo, le aplicaremos la propiedad "display:block" en la clase "block" a una imagen para que pueda responder al centrado con margenes automáticos

							h1 {
								display: inline-block;
							}

Le aplicamos la clase "block" y lo centramos con los margenes automáticos.

Como la imagen tiene un tamaño especifico que se lo da el tamaño propio de la imagen, el bloque no ocupa toda la pantalla y se comporta como si le ubieramos asignado un width fijo.
